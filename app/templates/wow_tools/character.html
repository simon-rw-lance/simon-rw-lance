WoW Tools

This is a tool used for obtaining ranked information from the video game World of Warcraft. In the game you can complete content known as "keys" which depending on the difficulty level and the speed of completion rewards you with an ammount of "score".

This tool allows you to enter your characters information (Name | Realm | Region) which will then send off an API request to the game servers. It will then return said information to our web server.

Currently this page will display to you your characters total score, along with the best key you have currently completed however there is a wealth of other additional information available to us that is simply not displayed.
